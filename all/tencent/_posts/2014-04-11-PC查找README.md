## 发布流程

先ARS预编译：
http://pub.qplus.oa.com/html/ars_precompile.php

得到类似地址：
/data/sites/cdn.qplus.com/qqfind/js/find.js
/data/sites/cdn.qplus.com/qqfind/index.html

进入ARS发布系统：
http://ars.isd.com/

1.类目选择：发布请求-免测版本发布
2.产品选择：开放平台 模块选择：公共平台 其他如版本号等可任意填写
3.把预编译得到的地址 粘贴到发布内容
4.下一部处理人选择： 测试童鞋的rtx，如alanqing
5.提交后知会测试
